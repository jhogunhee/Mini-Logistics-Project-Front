🏪 Convenience Store Inventory Manager

편의점 운영자의 관점에서
재고 관리와 WMS 개념을 재해석한 미니 프로젝트입니다.

복잡한 물류 관리가 아니라,
“지금 팔 수 있는 재고 상태”를 빠르게 판단하는 UI에 집중했습니다.

🎯 프로젝트 목적

기존 WMS의 관리·통제 중심 화면 ❌

편의점 운영자가 실제로 보는 흐름 ⭕

재고를 하나의 수량이 아닌

🧺 매대 재고 / 📦 창고 재고
로 분리해 관리하며,

실무 WMS의 할당 개념을 ‘매대 채우기’ 기능으로 재해석했습니다.
